ITEC 4610 - Project Development and Portfolio Defense

Project Tasks

Set up a Trello project with the following tasks. Share the project with your instructors.

  • Assign projects to teams
  • Do an internet search of the project and client
  • Formulate questions for client meeting
  • Contact client and set up initial meeting (Let instuctors know the meeting date)
  • Conduct meeting with client
  • Develop a Scope Document for the project.
  • Obtain Approval of Scope Document(Let instructor know the document has been approved)
  • Develop Client Communication Schedule and Change Management Form
  • Develop WBS, Estimates, Task Assignments (Place the WBS entities into Trello)
  • Each team will do a usability test on the other team's projects. (10/25/2022)
  • Project Presentations (11/01/2022)
  • Final Turnover of Project - Backup and Duplication (12/1/2022)
